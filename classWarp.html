Detailed Description

Represents a GPU warp.

The Warp class encapsulates the state and behavior of a GPU warp, including its ID, program counter, threads, active mask, and current instruction. A warp is a group of threads that execute instructions in lockstep. To be added: reconvergence stack to handle divergent control flow.
